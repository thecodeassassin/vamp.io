    Vamp 0.7.9: introducing the UI and CLI.

    We just pushed release 0.7.9 which is easily the biggest Vamp release to date! With this release we focused on making Vamp easy to install and work with.

    First of all, we package the first version of our graphical user interface. With this ReactJS based UI, you can create, update and delete Vamp artifacts quickly and dive into the Vamp metrics feed. Quick start →

    Secondly, we released the first version of the Vamp command line interface. The CLI allows admins and Devops engineers to do fully automated canary releases and blue/green deployments on Docker and Mesophere’s/DCOS Marathon from CI tools like Jenkins or just from bash scripts.

    You can install the CLI on your Mac using Homebrew, Learn more →

    Thirdly, we’ve updated our website, the configuration and installation documentation and quick start to make using and installing Vamp easier than ever. We support native package managers Centos, RHEL, Debian, Ubuntu. Check the docs →
